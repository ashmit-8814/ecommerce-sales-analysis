# 🛒 E-commerce Sales Analysis

## 📌 Project Overview

This project analyzes an e-commerce dataset to uncover sales trends, top-performing products, and regional performance.

## 🎯 Objectives

* Analyze sales performance by region
* Identify top-selling products
* Compare category-wise revenue
* Understand monthly sales trends

## 🛠️ Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Jupyter Notebook

## 📊 Key Insights

* West and North regions generate the highest revenue
* Electronics category dominates overall sales
* Laptops and mobiles are top-selling products
* Sales show an increasing monthly trend

## 💡 Business Recommendations

* Focus more on high-performing regions
* Increase inventory for electronics
* Target marketing in low-performing areas

## 📁 Files in Repository

* `analysis.ipynb` → Complete analysis
* `data.csv` → Dataset used
